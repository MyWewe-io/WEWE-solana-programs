### 3. **Maker Keypairs** (User-Generated)
- **Public Key**: Any (generated per maker)
- **Required For**:
  - `createProposal` - Create new token proposals
  - `createPool` - Create liquidity pools
  - `claimPoolFee` - Claim pool fees
- **Status**: Generated by users/makers

### 4. **Backer Keypairs** (User-Generated)
- **Public Key**: Any (generated per backer)
- **Required For**:
  - `depositSol` - Back proposals with SOL
  - `refund` - Request refunds
  - `claimMilestoneReward` - Claim milestone rewards
  - `airdrop` - Claim airdrops
  - `decrementBackerCount` - Decrement backer count
- **Status**: Generated by users/backers

### 5. **Payer Keypairs** (User-Generated)
- **Public Key**: Any (for paying transaction fees)
- **Required For**:
  - `mintSoulboundToUser` - Mint soulbound tokens
  - `airdrop` - Pay for airdrop transactions
  - Various operations requiring fee payment
- **Status**: Any wallet with SOL for fees

## üìç Program Derived Addresses (PDAs - No Keypairs Needed)

These are derived addresses controlled by the program, **no private keys required**:

### 1. **Vault Authority PDA**
- **Seed**: `"vault_authority"`
- **Purpose**: Controls token vaults
- **Derived**: Automatically by program

### 2. **Config PDA**
- **Seed**: `"config"`
- **Purpose**: Stores program configuration
- **Derived**: Automatically by program

### 3. **Mint Authority PDA**
- **Seed**: `"mint_authority"`
- **Purpose**: Controls soulbound token minting
- **Derived**: Automatically by program

### 4. **Maker Account PDAs**
- **Seed**: `["maker", maker_pubkey]`
- **Purpose**: Stores maker account data
- **Derived**: Per maker

### 5. **Proposal PDAs**
- **Seed**: `["proposal", maker_pubkey, proposal_index]`
- **Purpose**: Stores proposal data
- **Derived**: Per proposal

### 6. **Backer Account PDAs**
- **Seed**: `["backer", proposal_pubkey, backer_pubkey]`
- **Purpose**: Stores backer data per proposal
- **Derived**: Per backer per proposal
